Paquete de Datos¶

El paquete de datos se encuentra dedicado al manejo y extracción de datos desde servidores como desde archivos con formato especifico.

La finalidad del paquete es la estandarización de la obtención de datos, al igual que centralizar y simplificar la traducción de archivos y consultas SQL en objetos nativos de la librería.

Las módulos que se se encuentran en el paquete son:

  • Conexión: Clases dedicada a la conexión y obtención de datos abstractos desde servidores SQL.

  • Lector de Archivos: Clase dedicada a la lectura de datos desde archivos con formato estandarizado.

  • Lector Servidores: Clase dedicada a la lectura de datos desde los servidores internos.

Conexión¶

Este módulo está enfocado en la creación de conexiones a servidores SQL. Para esto contamos con la clase:

  • SQLConn: Clase dedicada al manejo de conexiones a servidores SQL.

class data.dbconnection.SQLConn(ip: str, database: str, username: Optional[str] = None, password: Optional[str] = None)¶

Conexión SQL

Se encarga de generar conexiones y ejecutar consultas a bases de datos

Parámetros:

  • ip (str.) – IP del servidor.

  • database (str.) – Nombre de la base de datos.

  • username (str.) – Nombre de usuario. (default None)

  • password (str.) – Password. (default None)

get_conn_string(database: str = '')¶

Obtener cadena de conexión

Genera una cadena de conexión standard utilizando los datos provistos. En caso no se haya provisto un nombre de usuario se genera una cadena por Trusted Connection.

Parámetros:

database (str.) – Base de datos a utilizar (opcional, en caso no se indique se utiliza la definida inicialmente).

Devuelve:

str con cadena de conexión
