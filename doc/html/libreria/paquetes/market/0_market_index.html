Paquete de Mercado¶

El paquete de mercado se encuentra dedicado a almacenar clases que representen información de mercado. Dentro de este paquete contamos con los siguientes módulos:

  • Calendarios: Clases para la generación y almacenamiento de datos relacionados a feriados y asociados a una localidad especifica.

  • Curvas: Clases dedicadas a la generación, almacenamiento y construcción de curvas de valorización y proyección.

  • Tipo de cambio: Clases de configuración y manejo de tipo de cambio.

  • Indices: Clases de configuración y manejo de indices de mercado.

  • Volatilidad: Clases dedicadas al almacenamiento, configuración y manejo de la volatilidades.

  • Datasets: Clase que permite almacenar, manejar y utilizar con todos los datos de mercado de manera conjunta.
