Funciones Base¶

Llamamos función base a toda aquella que se encuentra dentro del namespace de Finlib, estas funciones se encuentran «a la mano» debido a que son funciones (y clases) que alimentan a los demás módulos de la librería.

Dentro de estas funciones, y sin ningún orden en particular, tenemos:

  • Enumeradores: Clases estáticos con listados enfocados en estandarizar los códigos utilizados dentro de la librería.

  • Excepciones: Clases dedicadas a estandarizar los distintos errores que puede generar la librería.

  • Configuración: Clases dedicadas a reducir el tiempo de configuración con estándares de mercado o internos.

Enumeradores¶

Los enumeradores dentro de este código se utilizan para estandarizar la utilización de la librería entre los distintos usuarios y asi evitar que distintos programas que utilicen la librería utilicen parametrizaciones distintas.

Dentro de estos enumeradores tenemos:

  • Locality: Listado de localizaciones.

  • Currency: Listado de monedas.

  • MaxMin: Estandarización de representación de Máximo y Minimo.

  • FinancialPosition: Estandarización de posición larga o corta.

  • Periodicity: Listado de periodicidades.

  • BusinessDays: Listado de convenciones de días hábiles.

  • DayCount: Listado de convenciones de conteo de días.

  • Compounding: Listado de convenciones de composición de tasas.

  • InterpolationMethod: Listado de métodos de interpolación.

  • ExtrapolationMethod: Listado de métodos de extrapolación.

  • IndexType: Listado de tipos de índices financieros.

Este módulo contiene estandarizaciones de configuración para distintas clases.

class config.ConfigGenerator¶

Bases: object

Clase estática dedicada a la generación de diccionarios de configuraciones de otras funciones.

static generate_curve_config(interpolation_method=LOG_LINEAR, low_point_extrapolation_method=FLAT_RATE, high_point_extrapolation_method=RATE_SLOPE)¶

Genera un diccionario con el formato adecuado para configurar la clase Curve.

Parámetros

  • interpolation_method (InterpolationMethod.) – Método de interpolación. (default Log-Linear)

  • low_point_extrapolation_method (ExtrapolationMethod.) – Método de extrapolación en punto bajo. (default Flat Rate)

  • high_point_extrapolation_method (ExtrapolationMethod.) – Método de extrapolación en punto alto. (default Rate Slope)

Devuelve

Diccionario con configuración de curva
