.. _Módulo Excepciones:

**Módulo de Excepciones**
-------------------------

.. automodule:: exceptions

A continuación se presenta un listado de las excepciones contenidas en este módulo:

- :ref:`excepción_enumerador` (:class:`EnumException`): Maneja errores asociados a Enumeradores.
- :ref:`excepción_moneda` (:class:`CurrencyException`): Maneja errores asociados a Monedas.
- :ref:`excepción_fx` (:class:`FxRateException`): Maneja errores asociados a Tasas de Cambio.
- :ref:`excepción_calendario` (:class:`CalendarException`): Maneja errores asociados a calendarios de negociación.
- :ref:`excepción_curvas` (:class:`CalendarException`): Maneja errores asociados con curvas.
- :ref:`excepción_índices` (:class:`IndexException`): Maneja errores asociados con índices financieros.
- :ref:`excepción_dataset` (:class:`DatasetException`): Maneja errores asociados con Dataset.
- :ref:`excepción_portafolio` (:class:`PortfolioException`): Maneja errores asociados con Portafolio.
- :ref:`excepción_lectura_archivo` (:class:`FileReadException`): Maneja errores asociados con Lectura de Archivos.
- :ref:`excepción_query` (:class:`QueryException`): Maneja errores asociados con consultas a base de datos.
